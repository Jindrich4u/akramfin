Tough SDR Handheld Tactical Radio

Multi-band, multi-waveform radio for dismounted operations

Software-Defined Flexibility

The Tough SDR Handheld delivers tactical communications capability in a ruggedized form factor optimized for dismounted soldiers. Software-defined architecture supports multiple waveforms across VHF and UHF frequency bands, providing interoperability with coalition forces and legacy systems.

Integrated GPS, cryptographic security, and networking capabilities transform this radio into a comprehensive tactical communications and situational awareness tool. Intuitive user interface enables rapid configuration changes in high-stress operational environments.

Radio Capabilities

  • Frequency: 30-512 MHz
  • Power output: 0.1 to 5 Watts
  • Channel spacing: 25 kHz/12.5 kHz
  • Encryption: AES-256
  • Data rates: Up to 5 Mbps
  • GPS integrated
  • Push-to-talk voice
  • Text messaging

Supported Waveforms

  • SRW (Soldier Radio Waveform)
  • SINCGARS (legacy compatibility)
  • HAVE QUICK
  • Narrowband waveforms
  • Custom waveform loading

Ruggedization

  • MIL-STD-810G environmental
  • MIL-STD-461 EMI/EMC
  • IP67 water/dust ingress
  • Operating temp: -32°C to +60°C
  • Drop resistant: 1.5 meters
  • Salt fog and humidity tested

Dimensions

Battery

Interface

  • Audio: Speaker and headset jack
  • Data: USB and accessory ports
  • Programmable function keys
