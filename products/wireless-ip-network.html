Tactical Wireless IP Network System

Complete mobile ad-hoc networking solution for military operations

Brigade-Level Network Solution

The Tactical Wireless IP Network provides complete tactical communications infrastructure from individual soldiers through brigade headquarters. Integrated radios, servers, and management software deliver voice and data connectivity across dispersed forces in contested environments.

Self-forming mesh architecture maintains network operation despite node failures and electronic warfare. Quality of service and traffic management ensure mission-critical applications receive necessary bandwidth even under network stress.

Network Capabilities

  • Up to 20 Mbps data rates
  • Range: 5-50 km per hop

  • Voice and data simultaneity
